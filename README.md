# Student-Management-System-Using-JDBC
A web-based application built to manage student information using Java Servlets and Native JDBC. This project focuses on the core fundamentals of web development and manual database connectivity.

# Features
* Student Dashboard: View a complete list of registered students.
* Data Entry: Form-based student registration.
* Edit/Update: Modify student details using hidden IDs and SQL UPDATE queries.
* Delete Functionality: Remove records directly from the MySQL database.

# Tech Used
* Backend: Java (JDK 17)
* Web Server: Apache Tomcat
* Database: MySQL
* API: JDBC (Java Database Connectivity)
